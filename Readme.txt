User Management with Stripe is possible from within WordPress with Simple Stripe Checkout Manager.

== Description ==

The plugin is an add-on for Simple Stripe Checkout which allows you to view your user transactions within WordPress.

Instead of visiting your Stripe dashboard to review your transactions, you can do it from your WordPress admin area.

> "This plug-in will show Stripe transaction details such as customer name, purchase amount, date, and more within your WordPress admin area"

###Plugin Requirements###

This plugin requires the Simple Stripe Checkout plugin.[Get Simple Stripe Checkout](https://wordpress.org/plugins/stripe/)


###Available in the Pro version Only###

There is also a Pro version that works with Stripe Checkout Pro and contains the following shortcode support.

* **Integrates with WordPress user accounts** - Allow your users to view their transactions with integrated accounts created in WordPress.
* **Users can Cancel Subscriptions** - Allow your users to sign in and cancel their subscriptions.

== Installation ==

There are three ways to install this plugin.

= 1. Admin Search =
1. NOTE: Make sure that you have the Simple Checkout Pro installed.
1. In your Admin are, go to menu Plugins > Add.
1. Search for `Simple Stripe Checkout Manager`.
1. Find the plugin that's labeled `Simple Stripe Checkout Manager`.
1. Look for the author name `Kyle Brown` on the plugin.
1. Click to install.
1. Activate the plugin.
1. A new submenu item `Manager` will appear beneath the Simple Stripe Checkout menu.

= 2. Download & Upload =
1. Download the plugin (a zip file) on the right column of this page.
1. In your Admin, go to menu Plugins > Add.
1. Select the tab "Upload".
1. Upload the .zip file you just downloaded.
1. Activate the plugin.
1. A new submenu item `Manager` will appear beneath the Simple Stripe Checkout menu.

= 3. FTP Upload =
1. Download the plugin (.zip file) on the right column of this page.
1. Unzip the zip file contents.
1. Upload the `simple stripe checkout manager` folder to the `/wp-content/plugins/` directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. A new submenu item `Manager` will appear beneath the Simple Stripe Checkout menu.
